sweeps.sweep(loSource, udp, f_center, freqs, N_steps=500, freq_step=0.0)[source]

Actually perform an LO Sweep using valon 5009’s and save the data

Parameters:

  • loSource (valon5009.Synthesizer) – Valon 5009 Device Object instance

  • f_center – Center frequency of upconverted tones

  • freqs (List) – List of Baseband Frequencies returned from rfsocInterface.py’s writeWaveform()

  • udp (udpcap.udpcap object instance) – udp data capture utility. This is our bread and butter for taking data from ethernet

  • N_steps (Int) – Number of steps with which to do the sweep.
